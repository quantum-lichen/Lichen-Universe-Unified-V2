# Changelog - Lichen Universe Unified

Toutes les modifications notables de ce projet seront documentées dans ce fichier.
Le format est basé sur [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [2.1.0] - 2025-12-21 (The Great Refactor)
### Added
- **Manifest V2.1** : Source de vérité unique pour les Agents IA (`manifest.json`).
- **Lexique Unifié** : Définition stricte des symboles mathématiques ($S$ pour Entropie, $\mathcal{H}$ pour Harmonie).
- **MathJSON** : Toutes les formules sont maintenant parseables par machine.
- **Dependency Graph** : Arborescence explicite des composants dans le JSON.

### Changed
- **Nomenclature** : 
    - `H` (Entropie) devient `S`.
    - `H` (Harmonie) devient `ℋ`.
    - `Microkernel` devient `Lichen μ-Kernel`.
    - `FC496` devient `FC-496`.
- **Structure** : Fusion des anciens repos dispersés vers une architecture centralisée.

### Fixed
- Résolution des conflits de noms entre les théories thermodynamiques et quantiques.
- Correction des formules CEML et H-Scale pour utiliser les nouveaux symboles.

## [1.5.0] - 2025-12-20 (The Hack)
### Added
- **GitHub Pages Hack** : Utilisation de `manifest.json` pour contourner les filtres de contexte des LLMs.
- Première version agrégée des 50 repositories.

## [1.0.0] - 2024-01-01 (Genesis)
### Added
- Création initiale des concepts UICT, CEML et FC-496.
- Phase d'exploration et de prototypage rapide.
